# esahakulinen.fi

Personal portfolio and blog site built with Jekyll and hosted on GitHub Pages.

## Structure

- `_posts/` - Blog articles (Markdown)
- `_projects/` - Project showcases
- `_seo_pages/` - Programmatic SEO / Guide content
- `projects/` - Standalone interactive projects (React apps, demos, etc.)
- `assets/` - CSS, JavaScript, and images

## Adding Content

See [CONTENT_GUIDE.md](CONTENT_GUIDE.md) for detailed instructions on:
- Writing blog posts
- Adding projects
- Creating SEO pages
- Embedding React apps

## Quick Start

1. Edit files in the appropriate folder
2. Commit and push to GitHub
3. Site automatically builds and deploys

## Built With

- [Jekyll](https://jekyllrb.com/) - Static site generator
- [GitHub Pages](https://pages.github.com/) - Hosting
- Markdown - Content format
